Aadhaar is India's national identity number — a 12-digit unique identifier issued to over 1.35 billion residents by UIDAI. But how do you know if an Aadhaar number you've been given is actually in a valid format? This guide explains exactly that.

What Makes an Aadhaar Number Valid?

A valid Aadhaar number must satisfy three conditions:

  • Exactly 12 digits — no more, no less
  • Does not start with 0 or 1 — these digits are reserved by UIDAI
  • Passes the Verhoeff checksum — the 12th digit is mathematically derived from the first 11

The most important (and interesting) part is the Verhoeff algorithm — a checksum formula used by UIDAI to detect typos and transcription errors.

What is the Verhoeff Algorithm?

The Verhoeff algorithm was invented by Dutch mathematician Jacobus Verhoeff in 1969. Unlike simpler checksums (like Luhn used in credit cards), Verhoeff can detect:

  • All single-digit errors (typing 5 instead of 3)
  • All adjacent transposition errors (swapping 12 to 21)
  • Most twin errors (11 → 22)
  • Most jump transpositions (132 → 231)

UIDAI uses this algorithm when generating Aadhaar numbers. The 12th digit of every valid Aadhaar is specifically chosen so that when the full 12-digit number is run through the Verhoeff algorithm, it produces exactly 0. If even one digit is changed, the result will not be 0.

How Does the Validation Work — Step by Step?

Here's a simplified walkthrough of what happens when you click "Validate" on our Aadhaar tool:

  • Strip spaces from the input (UIDAI formats Aadhaar as 4-4-4 for readability)
  • Check the digit count — must be exactly 12
  • Check the first digit — must not be 0 or 1
  • Reverse the digits
  • Run through the Verhoeff multiplication and permutation tables
  • If the final value is 0 → Valid. Anything else → Invalid.

The entire process takes microseconds and needs no internet connection — it's pure mathematics.

What This Validator Cannot Tell You

It's important to be clear about what format validation cannot do:

  • It cannot tell you if the Aadhaar exists — UIDAI's database is not publicly accessible
  • It cannot tell you who the Aadhaar belongs to
  • It cannot verify biometrics or OTP
  • A valid format does not mean a valid person — someone could technically construct a format-valid Aadhaar number that was never actually issued

For official identity verification, always use UIDAI's authorised channels: the mAadhaar app, Digilocker, or UIDAI's official portal.

Common Mistakes That Cause Validation Failures

  • Including spaces or dashes in the number when entering manually
  • Confusing the digit 1 (one) with the letter I (capital i)
  • Misreading from a photocopy — especially digits 3/8 and 6/0
  • Using a VID (Virtual ID) instead of the Aadhaar number — VIDs are 16 digits and won't validate as Aadhaar

Using our validator is completely safe because the validation is entirely client-side. Your Aadhaar number never leaves your device. The JavaScript code runs locally in your browser — there is no server involved, no database, and no logging.
