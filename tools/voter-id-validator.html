Structure of a Voter ID (EPIC) Number

The standard EPIC number is a 10-digit alphanumeric code assigned by the Election Commission of India (ECI):

Characters Description
1 - 3 Functional Code: Usually represents the state or election registration code.
4 - 10 Serial Number: A 7-digit unique sequential number.

Older Voter IDs may have different formats, but most cards issued in the last decade follow this 3L + 7D structure.
