IFSC Code Validator

Verify if an IFSC (Indian Financial System Code) is in the correct format. Identify the bank name, confirm the mandatory '0' at position 5, and decode the branch code — all in your browser.

11 characters. 5th character must always be 0. Example: HDFC0001234

FAQs

IFSC (Indian Financial System Code) is used by India's electronic payment systems (NEFT, RTGS, IMPS) to uniquely identify a bank branch. It is required for all inter-bank money transfers in India.
Your IFSC code is printed on your cheque book, passbook, and bank statement. It also appears on your internet banking portal and is visible in banking apps. Every bank branch in India has a unique IFSC code.
The 5th character is reserved by RBI for future expansion. It is always '0' in all currently issued IFSC codes. This allows the RBI to use characters 1–4 for a future secondary bank code if needed.
