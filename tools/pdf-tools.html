What is the PDF command center?

The PDF command center is a browser-based workspace for common PDF actions such as merging, splitting, compressing, converting, rotating, locking, and organizing PDF files.

Are PDF files uploaded to VerifyDocs?

PDF actions are designed around privacy-first browser processing where supported. Sensitive files should not be sent by email or pasted into unknown tools.
