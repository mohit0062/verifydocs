Driving License Number Validator

Verify if an Indian Driving License (DL) number is in the correct format. Decode the state code, RTO code, and year of issue. Covers all 36 states and Union Territories.

Format: State (2) + RTO (2) + Year (4) + Serial (7). Example: MH0120201234567

Note: Format validation only. Does not verify with Sarathi (Parivahan) database.

DL Number Format

PartExampleDescription
Positions 1–2MHState code (2 uppercase letters)
Positions 3–401RTO (Regional Transport Office) code (2 digits)
Positions 5–82020Year of issue (4 digits)
Positions 9–151234567Serial number (7 digits)
