UPI ID (VPA) Validator

Validate the format of any UPI ID (Virtual Payment Address). Ensure the handle and PSP bank code follow the standard name@bank structure used by NPCI.

Example: success@upi, 9876543210@ybl, name.surname@okhdfcbank

What is a UPI ID?

A UPI ID, also known as a Virtual Payment Address (VPA), is a unique identifier used to send and receive money via the Unified Payments Interface. It acts as an email ID for your bank account.

Standard Structure:

handle @ psp_code

Common PSP Handles

  • @okaxis Axis Bank
  • @ybl Yes Bank
  • @okhdfcbank HDFC Bank
  • @okicici ICICI Bank
  • @upi NPCI Generic
  • @paytm Paytm
